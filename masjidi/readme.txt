Display prayer times from the Masjidi API for one or two masjids.

== Description ==

Display prayer and iqamah times for your masjid using the Masjidi API. Supports displaying times for two masjids side by side.

**Features:**

* Pull prayer times from Masjidi API
* Support for dual masjid display (side by side iqamah times)
* Automatic active prayer highlighting
* Jumu'ah times (up to 3)
* Iqamah change alerts
* Sunrise/sunset times
* Link to monthly calendar
* Clean, modern design
* Responsive layout

== Installation ==

1. Upload the `masjidi` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings → Masjidi to configure your masjid IDs
4. Add the shortcode `[masjidi_prayer_times]` to any page or post
5. Or use the "Prayer Times Widget" in Appearance → Widgets

== Configuration ==

1. Get your Masjid ID from MasjidiApp
2. Enter your Primary Masjid ID in Settings → Masjidi
3. Optionally add a Secondary Masjid ID for dual display
4. Customize the highlight colors

== Shortcode ==

Use `[masjidi_prayer_times]` to display the prayer times widget.
